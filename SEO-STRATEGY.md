### Core Web Vitals Optimization

| Metric | Target | Implementation |
|--------|--------|----------------|
| LCP (Largest Contentful Paint) | < 2.5s | Optimize hero images, use Next.js Image component |
| FID (First Input Delay) | < 100ms | Code splitting, reduce JS bundles |
| CLS (Cumulative Layout Shift) | < 0.1 | Set image dimensions, reserve space for dynamic content |
| INP (Interaction to Next Paint) | < 200ms | Optimize event handlers, use `useTransition` |
| TTFB (Time to First Byte) | < 600ms | Edge caching, optimize Supabase queries |
| FCP (First Contentful Paint) | < 1.8s | Inline critical CSS, preload key resources |
